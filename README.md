## Installation and Setup

1. **Create a virtual environment:**
   ```bash
   virtualenv venv

2. **Activate it and install requirements:**
   ```bash
   source venv/bin/activate
   pip install -r requirements.txt
   python -m spacy download en_core_web_sm

3. **Run app file:**
   ```bash
   python app.py
